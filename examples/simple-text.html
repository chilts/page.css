Simple Text

This page shows some simple text. It contains a <main> tag which itself contains one <section> tag and not much more. We've added a <footer> too, just to show you how that would work. The main content could be something generated from Markdown, Textile, or any other lightweight markup language, or just plain hand written HTML.

We try and space out each section so you don't have to worry about vertical spacing. The same with lists, pre, code, and other block-level elements so that they all just fit together nicely without you having to tinker. All you need to do is put the pagecss class on your body and that's it. This page also has the content class on the main tag to narrow and center the content.
